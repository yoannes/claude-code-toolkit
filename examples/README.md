## Usage

### Using Standalone Prompts

1. Open the prompt file you want to use
2. Copy the entire content
3. Paste into a Claude Code conversation
4. Claude will execute the audit/task

### Using Templates

Templates provide patterns you can adapt:

1. Copy the template content
2. Modify paths, project names, and sections for your project
3. Save to your own documentation directory
